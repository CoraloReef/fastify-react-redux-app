# fastify-react-redux-boilerplate

Boilertplate Node.js application with simple authorization

### Used

```sh
Fastify
Sqlite

React
Redux (Toolkit)

Pug
Bootstrap
SCSS

i18next
eslint
```

---

## Requirements

```sh
Heroku CLI
```

## Setup

```sh
$ make setup
```

## Run

```sh
$ make start
# Open localhost:5000
```
